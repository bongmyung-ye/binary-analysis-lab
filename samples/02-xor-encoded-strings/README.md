# 02 - XOR Encoded Strings

This sample stores a few strings in a simple XOR-encoded form and decodes them at runtime.

The goal is to practice finding decoding logic and comparing static string extraction with runtime behavior.

## Analysis focus

- Check which strings are visible before execution
- Locate the decoding function
- Identify the XOR key
- Observe decoded output at runtime
- Recreate the decoding logic with a small script if needed

## Build

```bash
gcc src/main.c -o bin/xor-encoded-strings.exe
```

From the repository root:

```bash
gcc samples/02-xor-encoded-strings/src/main.c -o samples/02-xor-encoded-strings/bin/xor-encoded-strings.exe
```
